## Project Overview
This project implements a supervised machine learning model to classify Iris flower species using Logistic Regression. The model predicts the species of a flower based on four physical features.

## Dataset
The Iris dataset contains 150 samples with four numerical features:
- Sepal length
- Sepal width
- Petal length
- Petal width

Target classes:
- Setosa
- Versicolor
- Virginica

## Algorithm Used
Logistic Regression
